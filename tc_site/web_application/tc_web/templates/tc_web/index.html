Time Candle

This is a very simple task-tracker application that can help you to manage your time. User friendly interface provides you to create tasks and projects and use them very fast with your friends! Many people are already tested this app, you may be the next!

Versions

In current state we have a web application that you are using right now, and a console application, that you may find in the lowest link below. Also here you may find API for Time Candle and use it for free in any cases.

Guidelines

Also Time Candle has querying algorithms that provides a tool for advanced task search. Use the following filters for it: Parents: tasks with specified parents. Tids: tasks with specified tids Receiver: tasks with specified receivers pay ids Titles: Strings for tasks titles Supported operators: AND, OR, (, ). Special filters: Parents: None - prints only personal tasks Examples: (parents: 1 5 3 OR tids: 4 6 7) AND titles: new work candy
